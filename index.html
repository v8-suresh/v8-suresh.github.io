   Hadoop Apps
   Technologies :

Cloudera CDH 5.1.3 Hadoop 2.1.3 MR Unit 0.9.0 Apache pig Apache Hive Apache HBase

Apache Spark Apache Solr Amazon EC2

   Description :

This is a multi module maven Application developed to learn Hadoop and its eco system components. Every module is a Hadoop Application which uses different Hadoop Eco system components such as Simple Hadoop Application, HBase, Spark, Pig, Hive. This application was developed to execute on top of 8 node cluster running on Amazon EC2 infrastructure. Hadoop Cluster is managed by Cloudera Distribution for Hadoop (CDH Version 5.1.3). This application uses GSOD weather data-set which is of size 20 GB which is pulled from Amazon Public data-set.
